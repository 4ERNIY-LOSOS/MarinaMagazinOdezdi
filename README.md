# Marina Magazin Odezdi

Это десктопное WPF-приложение для управления интернет-магазином одежды. Приложение разработано на .NET и использует базу данных SQL Server для хранения данных.

## Основные возможности

- Просмотр каталога товаров с фильтрацией и сортировкой.
- Регистрация пользователей и аутентификация.
- Корзина для покупок.
- Оформление заказов.
- Просмотр истории заказов в профиле пользователя.
- Панель администратора для управления товарами, категориями и пользователями.

## Вход в панель администратора

Для доступа к панели администратора используйте следующие учетные данные:

- **Email:** `admin@shop.com`
- **Пароль:** `admin`
