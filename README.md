# Marina Magazin Odezdi

Это десктопное WPF-приложение для управления интернет-магазином одежды. Приложение разработано на .NET и использует базу данных SQL Server для хранения данных.

## Настройка базы данных

Перед первым запуском приложения необходимо создать и заполнить базу данных. Для этого у вас должен быть установлен SQL Server (например, бесплатная версия Express Edition).

1.  Откройте терминал (Command Prompt или PowerShell).
2.  Перейдите в корневую папку проекта.
3.  Выполните следующие команды по очереди:

    ```shell
    # Создание структуры базы данных и таблиц
    sqlcmd -S .\SQLEXPRESS -E -i setup_database.sql

    # Заполнение базы данных тестовыми данными (товары, категории, админ)
    sqlcmd -S .\SQLEXPRESS -E -i seed_database.sql
    ```

    **Примечание:** Если ваш экземпляр SQL Server называется иначе, чем `SQLEXPRESS`, замените `.\SQLEXPRESS` на правильное имя вашего сервера (например, `.` или `localhost`, если это экземпляр по умолчанию).

## Основные возможности

- Просмотр каталога товаров с фильтрацией и сортировкой.
- Регистрация пользователей и аутентификация.
- Корзина для покупок.
- Оформление заказов.
- Просмотр истории заказов в профиле пользователя.
- Панель администратора для управления товарами, категориями и пользователями.
